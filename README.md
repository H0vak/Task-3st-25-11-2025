Custom String Replacement Logic (C++)
Цей проєкт демонструє навички низькорівневої маніпуляції рядками через порівняння власної реалізації алгоритму заміни підрядка зі стандартним методом бібліотеки STL.

Технології:
C++.

String Manipulation: Реалізація власної логіки конкатенації та зрізів рядків через цикли.

STL (Standard Template Library): Використання методу st1.replace() для валідації результатів.

Бібліотека <windows.h>: Для коректної підтримки кириличного тексту в консолі.

Що робить ця програма:
Алгоритмічна заміна: Функція replaceCustom вручну перезбирає рядок, розділяючи його на три частини: префікс до позиції заміну, новий підрядок та суфікс після видаленого фрагмента.

Порівняльний аналіз: Програма виводить два результати — отриманий власноруч та отриманий через std::replace, що дозволяє переконатися в ідентичності логіки.

Керування потоком вводу: Використано cin.ignore(), що демонструє розуміння особливостей роботи буфера при зміні типів вводу (з int на getline).

Гнучкість параметрів: Користувач може вказати будь-яку точку старту та кількість символів для видалення, що тестує стійкість алгоритму до різних індексів.
